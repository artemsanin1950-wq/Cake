# ProjectX — расчёт индивидуального рациона питания

Выпускная квалификационная работа: **«Разработка веб-приложения для автоматизированного расчёта индивидуального рациона питания с интеграцией данных сервиса доставки продуктов»**.

## Идея и назначение

Веб-приложение помогает пользователю рассчитать индивидуальный рацион питания (ориентируясь на параметры пользователя и цели), а также сопоставить рацион с товарами из сервиса доставки продуктов (интеграция каталога/цен/наличия) для упрощения заказа.

## Стек технологий (зафиксированный)

- **Языки**: TypeScript (frontend), Python (backend)
- **Платформа**: Web (клиент) + Backend API
- **Frontend**: React
- **Backend**: FastAPI
- **Инструменты разработки**:
  - **VCS**: Git (публикация в GitHub/GitLab)
  - **Сборка**: Node.js + npm (frontend), pip/venv (backend)
  - **Тестирование**: (будет добавлено по мере реализации)

> Использование технологий за пределами данного стека — только с обоснованием.

## Архитектура проекта

Подход: **Layered Architecture (слоистая архитектура)**.

- **frontend (UI слой)**: интерфейс, маршрутизация/состояние, взаимодействие с API.
- **backend (API слой)**: HTTP API, валидация входных данных, выдача результатов расчётов.
- **backend/services (слой доменной логики/сервисов)**: расчёт рациона, преобразование данных, интеграция с внешними сервисами доставки (будет развиваться).

Взаимодействие компонентов (в общих чертах):

1. Пользователь работает с UI (React).
2. UI вызывает методы Backend API (FastAPI).
3. Backend использует сервисы из `backend/services` для расчёта/интеграций и возвращает результат UI.

## Запуск проекта (стартовый каркас)

На текущем этапе репозиторий содержит **только базовую структуру** и пустые точки входа.

### Backend (FastAPI)

Планируемый запуск (после добавления зависимостей и кода приложения):

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install fastapi uvicorn
uvicorn backend.main:app --reload
```

### Frontend (React)

Планируемый запуск (после инициализации приложения, например Vite/CRA):

```bash
npm install
npm run dev
```
