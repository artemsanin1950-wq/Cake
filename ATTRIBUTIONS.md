# HealthyMeal - Приложение для расчета рациона

Данный репозиторий содержит начальный этап реализации фронтенд-части приложения HealthyMeal.

## 🎨 Дизайн и макет (UI/UX)
Интерфейс приложения спроектирован в Figma. 
⚠️ **https://morph-whale-14626676.figma.site/meal-plan**

### Назначение макета:
Визуализация пользовательского интерфейса, проектирование расположения элементов (layout) и отработка базовых сценариев взаимодействия пользователя с сервисом по подбору индивидуального питания и заказу продуктов.

### Реализованные экраны:
1. Главная страница (презентация сервиса).
2. Страница калькулятора ИМТ (форма ввода параметров).
3. Каталог "Рацион" (карточки продуктов с разбивкой по приемам пищи).
4. Корзина (список покупок и расчет итоговой стоимости).
5. Страницы Регистрация/Логин.

### Отраженные пользовательские сценарии:
- **Расчет показателей:** Ввод данных пользователем -> получение рекомендаций.
- **Сбор рациона:** Просмотр КБЖУ -> выбор продуктов по приемам пищи -> добавление товаров в корзину.
- **Оформление заказа:** Проверка корзины -> просмотр стоимости доставки -> переход к оформлению заказа.

## 💻 Текущий статус разработки (Начальный этап)
В рамках текущего этапа реализовано:
- Базовая навигация (роутинг) между экранами.
- Каркас основных страниц (layout, Header).
- Подключены архитектурные слои (UI-компоненты отделены от страниц).
- Приложение собирается и запускается без ошибок. Бизнес-логика на данный момент использует моковые (тестовые) данные.
